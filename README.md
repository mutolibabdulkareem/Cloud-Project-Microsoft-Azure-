Azure Cloud Engineering Lab Portfolio

📌 Overview
This project showcases a hands-on cloud engineering lab built on Microsoft Azure. It demonstrates the deployment of secure infrastructure, web applications, backend services, and serverless workflows, simulating real-world enterprise cloud environments.

🧱 Architecture
This lab includes multiple components:
Virtual Network (VNet) with subnet segmentation
Network Security Groups (NSG) for traffic control
Windows Server Virtual Machine (VM)
Web Application hosted on Azure App Service
API + Azure SQL Database integration
Blob Storage with Azure Functions (serverless automation)

🚀 Implemented Solutions
1. 🔐 Secure Infrastructure Deployment
Designed and deployed a secure virtual network with multiple subnets
Configured NSG rules to restrict RDP access to a specific public IP
Provisioned and managed a Windows Server VM
Implemented network segmentation and access control

2. 🌐 Web Application Deployment
Deployed a web application using Azure App Service
Configured application hosting and environment settings
Verified public accessibility and application performance

3. 🔗 API & Database Integration
Built and deployed a simple API on Azure App Service
Integrated Azure SQL Database for backend data storage
Configured secure database connectivity and basic operations

4. ⚡ Serverless Storage Workflow
Implemented event-driven architecture using Blob Storage and Azure Functions
Configured triggers to automatically process uploaded files
Demonstrated serverless automation and resource efficiency


🔐 Security Implementation
Restricted RDP access using IP-based filtering
Applied NSG rules to control inbound and outbound traffic
Segmented network resources using subnets
Followed least-privilege access principles

🧹 Cost Management
All resources were deployed for testing purposes and deallocated or deleted after use to minimize costs.
